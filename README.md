# Hackcelerate 2025 Voting System

A PHP-based voting system for hackathon teams and problem statements.

## Setup Instructions

1. **Database Setup:**
   - Import `hackcelerate_voting.sql` to your MySQL database
   - Create database: `hackcelerate_voting`

2. **Configuration:**
   - Copy `config-sample.php` to `config.php`
   - Update database credentials in `config.php`

3. **Run the application:**
   - Access `vote.php` through your web server

## Features
- Team voting system
- Problem statement voting
- Admin results view
- Real-time vote counting

## Database Structure
See `hackcelerate_voting.sql` for complete schema.
